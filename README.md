# Quote Generator

A simple and clean quote generator built with HTML, CSS, and JavaScript.  
It displays random motivational quotes with a minimal UI and smooth interaction.

This project was created to practice JavaScript logic, DOM manipulation, and basic project structure.

---

## Features

- Generates random quotes on each action  
- Clean and minimal user interface  
- Responsive layout for different screen sizes  
- Fast and lightweight with no external libraries  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---


## How It Works

- Quotes are stored in JavaScript
- A random quote is selected using JavaScript logic
- DOM manipulation updates the quote text dynamically

---

## Purpose of This Project

This project was built as a hands-on practice project to:
- Strengthen JavaScript fundamentals
- Improve understanding of DOM manipulation
- Build confidence in creating small interactive web apps
